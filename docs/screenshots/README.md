# Screenshot Placeholders

Capture real UI screenshots after running the app locally and save them in this folder using these filenames:

- `home.png`
- `suggestions-iph.png`
- `suggestions-py.png`
- `cache-hit.png`
- `trending.png`
- `metrics-dashboard.png`
- `cache-routing.png`

These files are not included by default because they should reflect the live local run of the current project state.
