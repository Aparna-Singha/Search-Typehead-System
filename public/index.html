Search Typeahead System

Type a prefix to fetch suggestions from the in-memory Prefix Index. Submitting a search queues a batched write and updates recent trending activity in Redis.

Popular suggestions are shown when the input is empty.

      How this demo behaves

      • Suggestions are debounced before each API call.
      • Redis caches repeated prefix lookups for faster reads.
      • Search submissions return quickly and write to PostgreSQL in batches.
      • New activity can appear in trending before the next database flush finishes.
